Oculus for Trentino

Installing Guide:

1) download and install Oculus SDK for Mac or Windows 
	https://developer.oculus.com/downloads/#sdk=any

2) download and install Oculus Runtime for Mac or Windows
	https://developer.oculus.com/downloads/#sdk=any

3) open RiftConfigUtil -> select Gender and Player Height

4) run the application Oculus4Trentino and enjoy!!


Requisites:

Monitor resolution higher than 1440x900
